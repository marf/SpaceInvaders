void GenerateFreeCoord ( int &  x,
Game &  game 
)

Recursive function which generates an x free coord for the enemy using a delta (ENEMY_DELTA_SPAWN)

Parameters
[in]xis used to return the x free cord when we have found it
[in]gameis used to check if other enemies seize the position (interval from x-ENEMY_DELTA_SPAWN to x+ENEMY_DELTA_SPAWN)

The recursion ends when we find a free x coord so that the enemy hasn't others enemies around it in an interval with a radius of ENEMY_DELTA_SPAWN. The function uses rand() (we have set the random seed in InitGame) to create a random x coord.
