void Fire ( Player &  player)

Fires a bullet if we have finished reloading.

Parameters
[in]playeris used to check if we are ready to fire and to add a bullet

We check if we are ready to fire by subracting the current time to the next time we will be ready to fire. If the result is > 0 it means that player.nextFire is in the past and the reloading process is finished. When we fire we set the player.nextFire time to the current time + PLAYER_FIRE_RATE which is the time we will spend for the reloading process.
